# web-webserver

A simple multithreaded web server built with Rust, capable of serving HTML files and handling basic HTTP requests.

## How to Run

1. Make sure you have [Rust](https://rust-lang.org/tools/install/) installed.
2. Clone this repository and navigate to the project directory.
3. Run the web server with:

   ```sh
   cargo run
   ```

4. Open your browser and navigate to `http://127.0.0.1:3000` to view the server's response.
